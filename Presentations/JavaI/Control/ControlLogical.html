Logical Operators

Logical operators allow us to combine multiple boolean values to produce a single boolean value.
&&
And
Returns a true value if both expressions are true.
This operator only evaluates the second expression if necessary.
||
Or
Returns a true value if either expression is true.
This operator only evaluates the second expression if necessary.
&
And (bitwise and)
Returns a true value if both expressions are true.
This operator always evaluates both expressions.
For interger types, this actually calculates a binary 'and' of the two operands. (1010 & 0110) evaluates to what?
|
Or (bitwise or)
Returns a true value if either expressions is true.
This operator always evaluates both expressions.
For integer types, this actually calculates a binary 'or' of the two operands. (1010 | 0110) evaluates to what?
!
Not
Reverses the value of the expression.
