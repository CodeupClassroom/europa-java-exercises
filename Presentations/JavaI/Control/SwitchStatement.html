Switch Statement

Switch statements can be thought of as a special case of the if-else if block
In Java, all values in the case labels of a given switch statement must be of the same type.
They must also be either literals, or final variables.
A switch works with the byte, short, char, and int primitive data types.
It also works with enumerated types.
And it works with the String class, and certain wrapper types: Character, Byte, Short, and Integer
Include a break statement inside of each case block.
